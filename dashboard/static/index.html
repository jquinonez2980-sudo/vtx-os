How to connect Gmail
1. Open a terminal in your AcumenAI project directory
2. Run: python scripts/gmail_auth.py
3. Follow the browser prompts to authorize Gmail access
4. Click Check Status above once complete
This grants read-only Gmail access to detect bank statement emails. Credentials are stored in your local Secret Manager or config folder.
